## 提炼原则

- **从具体到抽象**：品味必须从具体代码示例中归纳，不能凭空臆造
- **保留原话**：用户说出的关键判断词句是品味的直接体现，原样保留
- **三级分类**：原则（不可违背）、偏好（推荐风格）、反模式（明确否定）
- **标注理由**：如果用户解释了为什么，必须附上理由
- **标注来源**：每条品味标注来源于哪次 review
