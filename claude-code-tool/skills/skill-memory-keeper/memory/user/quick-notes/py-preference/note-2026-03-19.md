## 记录详情

### pref-001: For 循环变量重用

- **场景**: 同一函数中使用多个 for 循环，循环变量同名导致 mypy 报类型不兼容错误
- **决策**: 使用不同变量名（如 item、calc_item）而非类型注解或 type: ignore
- **提炼原则**: 语义优先、显式优于隐式、工具服务于人

### pref-002: mypy 类型错误的正面修复

- **场景**: 第三方库（如 SQLAlchemy）类型存根不完整导致 mypy 报错
- **决策**: 创建辅助方法使用 cast 正面修复，而非使用 type: ignore
- **提炼原则**: 正面修复优于忽略、集中管理重复逻辑、语义化命名
